# Logistics AI Connector

A web-based logistics tool that connects to a spreadsheet and integrates AI for data processing.

## Features
* **AI Handshake:** Processes data using [meta-llama/llama-3.3-70b-instruct:free].
* **Live Database:** Synchronizes directly with a spreadsheet.
* **Responsive UI:** Built with HTML and CSS.
